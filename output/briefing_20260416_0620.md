### 2. Claude Code Is Shipping at an Extraordinary Pace

Claude Code pushed **at least 10 releases in 8 days** (April 8--15), an unusually compressed cadence for a developer tool with 114,000+ GitHub stars. The features tell a story of rapid maturation:

- **`/tui` command** (v2.1.110, Apr 15): A full-screen, flicker-free terminal UI mode -- a significant quality-of-life improvement for long sessions.
- **1-hour prompt cache TTL** (v2.1.108, Apr 14): `ENABLE_PROMPT_CACHING_1H` environment variable, available across API, Bedrock, Vertex, and Foundry. Longer caches mean meaningfully lower token costs on repetitive workloads.
- **PreCompact hooks** (v2.1.105, Apr 13): Hooks can now block context compaction, giving developers control over when Claude summarizes long conversations -- critical for agentic workflows where compaction can lose state.
- **`/team-onboarding`** (v2.1.101, Apr 10): Generates a teammate ramp-up guide from local usage patterns. Signals Anthropic is targeting team-wide Claude Code adoption, not just individual developers.
- **Vertex AI setup wizard** (v2.1.98, Apr 9): Interactive GCP authentication flow for enterprise users -- lowering the barrier to enterprise cloud deployment.

The open issue count (10,158) is high for a project this size, suggesting either very active user engagement or a product moving faster than its support capacity. Worth watching.

### 3. Claude Managed Agents -- A New Product Surface Emerges
